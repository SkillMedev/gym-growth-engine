# Gym Growth Engine

**The operator stack to fill, sell, and scale a gym** — built in-house by [Skill&nbsp;Me](https://skillme.dev).

A pack of 12 self-contained Claude skills operationalizing the offer, lead-gen,
sales, and money-model frameworks behind Gym Launch and Acquisition.com. Model
unit economics and Client-Financed Acquisition, build a Grand Slam Offer and
premium pricing, fill the gym with a transformation challenge and Meta ads,
close with the CLOSER consult and beat objections, kill churn, drive referrals,
and run it all on a KPI scoreboard. Every skill ships with runnable calculators
and ready-to-use templates.

⭐ **If this is useful, star the repo** — it's how we gauge what to build next.

## Install

- **From the catalog:** [skillme.dev/pack/gym-growth-engine](https://skillme.dev/pack/gym-growth-engine) — install the whole pack into Claude in one step.
- **With the skills CLI:** `npx skills add aouellets/gym-growth-engine`
- **Manually:** copy any `skills/<slug>/SKILL.md` into your Claude skills directory.

## Skills in this pack

- **[Gym Money Model](skills/gym-money-model/SKILL.md)** — Use when a gym owner needs to model unit economics or decide whether growth pays for itself.
- **[Grand Slam Offer Builder](skills/grand-slam-offer-builder/SKILL.md)** — Use when a gym owner needs to design or sharpen a core offer so prospects feel stupid saying no.
- **[Gym Pricing and Guarantees](skills/gym-pricing-and-guarantees/SKILL.md)** — Use when a gym owner is setting prices or designing a guarantee.
- **[Core Four Lead Engine](skills/core-four-lead-engine/SKILL.md)** — Use when a gym owner needs more leads or a plan to get known.
- **[Gym Transformation Challenge](skills/gym-transformation-challenge/SKILL.md)** — Use when a gym owner is building or running a transformation challenge as a front-end offer.
- **[Gym Meta Ads Funnel](skills/gym-meta-ads-funnel/SKILL.md)** — Use when a gym owner is running or planning Meta (Facebook/Instagram) ads to fill a challenge or front-end offer.
- **[CLOSER Sales Script](skills/closer-sales-script/SKILL.md)** — Use when a gym owner or coach runs the intro consult or wants to close more consults.
- **[Objection Handling and Speed to Lead](skills/objection-handling-and-speed-to-lead/SKILL.md)** — Use when a gym owner is losing leads to slow follow-up or to objections.
- **[Retention and Churn Killer](skills/retention-and-churn-killer/SKILL.md)** — Use when a gym owner is losing members or wants to keep them longer.
- **[Referral and Affiliate System](skills/referral-and-affiliate-system/SKILL.md)** — Use when a gym owner wants others to bring in members.
- **[KPI Scoreboard and Cadence](skills/kpi-scoreboard-and-cadence/SKILL.md)** — Use when a gym owner wants to run the business on numbers.
- **[Hiring A-Players and SOPs](skills/hiring-a-players-and-sops/SKILL.md)** — Use when a gym owner is hiring or wants to systematize the business.
